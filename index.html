Estrutura básica do HTML

A estrutura básica do HTML é composta por várias tags que organizam e definem o conteúdo da página web. São elas:

  1. < ! DOCTYPE html > : Declara que este documento é um documento HTML5.
  2. < html lang="pt-br" > : A tag raiz do documento HTML, com o atributo lang especificando o idioma da página.
  3. < head > : Contém metadados sobre o documento, como o charset, título e informações de viewport.
  4. < meta charset="UTF-8" > : Define a codificação de caracteres do documento.
  5. < meta name="viewport" content="width=device-width, initial-scale=1.0" > :Define as configurações de visualização para dispositivos móveis.
  6. < title > Título da Página < / title > : Define o título da página, que aparece na aba do navegador.
  7. < body > : Contém todo o conteúdo visível da página.
  8. < header > : Define a seção de cabeçalho da página, geralmente contendo o título principal.
  9. < main > : Contém o conteúdo principal da página.
  10. < footer > : Define a seção de rodapé da página.

Atributos das Tags

Cada Tag HTML podem ter atributos globais e específicos. Estes atributos podem ter informações adicionais sobre a Tag ou definem como ela deve ser exibida. Tais atributos são colocados dentro da Tag de abertura que na maioria das vezes consistem em um nome de atributo e um valor. Ex:

Atributos Globais

  • Id = (Identificador de elemento) Quando trabalhado no JavaScript, essa linguagem tem como ler o html e pegar um elemento específico e trocar o conteúdo dele via linha de código por exemplo. Se clicar no botão ele altera o texto. Serve para identificar bo JavaScript qual elemento se quer alterar
  • Style = colocar comandos no css direto no elemento. Serve para formatar deixar bonito.
  • Class = padronizar tudo que tiver título principal na página ficar de uma cor ou no formato que eu predefinir.

Atributos Específicos

Tag < a > (âncora/hiperlink)

  • href: Especifica a URL para a qual o link aponta.
  • target: Define onde abrir o link (ex. _blank para abrir em uma nova aba).
  • rel: Especifica a relação entre o documento atual e o linkado (ex. nofollow).

Tag < img > (imagem)

  • src: Especifica o caminho da imagem.
  • alt: Fornece um texto alternativo que descreve a imagem.
  • width e height: Define a largura e a altura da imagem.

Tag < form > (formulário)

  • action: Define a URL para onde os dados do formulário são enviados.
  • method: Define o método de envio (GET ou POST).

    • Textos

    há várias tags que você pode usar para estruturar e formatar textos. Aqui estão algumas das principais tags de texto em HTML:

    Tags de Títulos

    • < h1 > a < h6 >: Definem títulos, onde < h1 > é o título mais importante e < h6 > o menos importante.

    Tags de Paragrafos

    • < p >: Define um parágrafo.

    Tags de Formatação de Texto

    • < b > ou < strong > : Deixa o texto em negrito.
    • < i > ou < em > : Deixa o texto em itálico.
    • < u > : Sublinha o texto.

    Tags de Listas

    • < ul > : Define uma lista não ordenada (com marcadores).
    • < ol > : Define uma lista ordenada (numerada).
    • < li > : Define um item de lista.

    Outras Tags Úteis

    • < a > : Cria um hiperlink.
    • < br > : Insere uma quebra de linha.
    • < hr > : Insere uma linha horizontal.

    Listas ordenadas e não ordenadas

    ol= lista ordenada. ul= lista não ordenada. A ul não utiliza números, ao contrário da ol que possui em sua sequência números.
